## Hello World

```kotlin
fun main() {
	println("Hello World!")
}
```

`print()` prints a line; `println()` prints a line and ends it with a new line symbol

## Variables

```kotlin
val age = 5
val name = "Rover"
```

```kotlin
println("Happy birthday, ${name}!")
// output: Happy Birthday, Rover!
```
